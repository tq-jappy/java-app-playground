# java-app-playground

- Java 8
- Spring Boot 2.1.1.RELEASE
- JPA & JOOQ

## Setup

```
$ docker-compose build
$ docker-compose up -d
```

## Migration

```
$ ./gradlew flywayMigrate
```

## JOOQ CodeGen

```
$ ./gradlew generateMainJooqSchemaSource
```

## Start Application

```
$ ./gradlew bootRun
$ open http://localhost:8080/
```
